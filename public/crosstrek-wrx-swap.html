2019 Crosstrek VB WRX Swap

Purpose: a practical technical plan for building a VB WRX-powered Crosstrek from a 2019 Crosstrek recipient and a complete VB WRX donor.

Method: move the WRX drivetrain and nervous system into the Crosstrek shell, then adapt hatch-specific body wiring, cooling, SRS, exhaust, and suspension.

Core instruction: buy the whole WRX donor. Keep the wiring, modules, keys, dash structure, subframes, pedals, SRS components, and every connector. A loose engine is not a build plan.
Recipient
2019 Crosstrek Premium
Donor
2022+ VB WRX 6MT
Engine
FA24 turbo
Trans
6-speed manual
Architecture
WRX ECU / BCM / harness
Custom zones
Hatch wiring, cooling, SRS, exhaust
First version should be OEM-first. Get it starting, scanning cleanly, cooling properly, and driving before tuning or adding power.

Parts to retain

Strip the donor like every small bracket and connector matters, because it does. The WRX donor supplies the systems; the Crosstrek supplies the hatch body.

Keep from VB WRX

  • FA24 turbo engine assembly, turbo, intake/intercooler plumbing
  • 6MT, clutch/flywheel, hydraulics, pedal box, trans mounts
  • Front subframe with drivetrain attached where possible
  • Rear subframe, suspension, brakes, hubs, axles as required
  • ECU, BCM, key/immobilizer components
  • Bulkhead, engine, dash, body, door, SRS harness sections
  • Fuse boxes, relay panels, cluster, dash beam, steering column, switchgear
  • Airbag module, crash sensors, pretensioner wiring, SRS pieces
  • Radiator, fans, intercooler path, front clip/core support if doing WRX nose
  • Hood, bumper, headlights, fenders, brackets, clips, supports

Keep from Crosstrek

  • Hatch shell, doors, hatch, cargo structure
  • Rear lamps and lamp pigtails if retaining Crosstrek rear lights
  • Rear hatch release hardware
  • Rear wiper motor, washer pump, washer tank, stalk as needed
  • Backup camera hardware and connectors if adapting Crosstrek camera
  • Rear window regulators and door internals if WRX regulators do not fit
  • Rear body connectors/pigtails for hatch wiring adaptation
  • Rear trim/cargo pieces
  • Crosstrek front body only if not doing WRX nose conversion

Build setup

  • Heavy-duty lift table for engine/trans/front-subframe assemblies
  • Wheel dollies for non-steerable shells
  • Label printer / masking tape / paint pens
  • Bag-and-tag hardware system
  • Connector photo log before disassembly
  • Spreadsheet for donor parts, transferred parts, discarded parts
  • Factory service manuals and wiring diagrams for both vehicles
  • Subaru-capable scan/programming tools or shop access

Build sequence

The order follows the documented VB Crosstrek build: strip both cars, transplant the WRX electrical backbone, move the drivetrain/subframes, then solve hatch-specific wiring and finishing systems.

Strip the WRX donorPreserve the complete WRX nervous system.toggle
  • Remove seats, dash, console, steering column, pedals, dash beam, door cards, carpet.
  • Extract ECU, BCM, cluster, fuse/relay boxes, key/immobilizer, switchgear, SRS modules.
  • Remove and label engine, bulkhead, dash, body, door, and SRS harnesses.
  • Remove drivetrain, subframes, brakes, hubs, suspension, radiator/intercooler, front clip parts.
Strip the Crosstrek recipientExpose the shell for WRX harness and drivetrain installation.toggle
  • Remove interior enough to route WRX harness and install WRX dash beam/modules.
  • Remove Crosstrek drivetrain/front subframe assembly.
  • Remove rear subframe/suspension if the chosen architecture uses WRX rear hardware.
  • Remove front core support/body pieces if doing WRX nose conversion.
  • Retain rear-body pigtails for lamps, hatch release, rear wiper, camera, and windows.
Install WRX electrical backboneMake the Crosstrek shell electronically WRX-first.toggle
  • Lay WRX bulkhead/main harness into the Crosstrek shell.
  • Install WRX BCM, ECU path, dash beam, steering column, pedals, cluster, fuse/relay boxes.
  • Route WRX harness branches before closing interior.
  • Keep Crosstrek rear-body connectors available for hatch-function adaptation.
Install WRX drivetrain/front subframeTransfer FA24/6MT as an assembly.toggle
  • Drop Crosstrek engine/trans/front subframe with lift table.
  • Prepare WRX FA24/6MT/front-subframe assembly.
  • Raise WRX assembly into Crosstrek shell and bolt in.
  • Check firewall, turbo, steering, axle, mount, and cooling clearances.
Transfer rear suspension/brake architectureFollow the donor architecture far enough to keep drivetrain and braking coherent.toggle
  • Move WRX rear subframe/suspension/brakes/hubs as needed.
  • Resolve wheel bolt pattern and hub architecture.
  • Keep alignment and tire-clearance decisions tied to final rally/off-road stance.
Solve body and coolingChoose WRX nose conversion or Crosstrek nose with custom cooling.toggle
  • WRX nose path: weld WRX core support into Crosstrek rails, test-fit radiator, hood, bumper, headlights.
  • Address fender mismatch: WRX fender does not bolt cleanly to Crosstrek door/body line.
  • Crosstrek nose path: design custom front-mount intercooler/cooling package around Crosstrek bodywork.
Adapt hatch wiringThe WRX is a sedan. The Crosstrek rear body is the custom zone.toggle
  • Lengthen and repin rear lamp wiring.
  • Lengthen/repin hatch release wiring.
  • Lengthen/repin backup camera wiring.
  • Wire rear wiper/washer through BCM programming or a custom bypass harness.
  • Wire Crosstrek rear regulators from WRX master window switch if WRX rear regulators do not fit.
  • Verify locks, remote functions, lights, camera, hatch, rear windows, rear wiper.
Solve SRSDo not close the dash until SRS is clean.toggle
  • Install WRX airbag module, sensors, and SRS harness strategy.
  • Resolve rear seatbelt pretensioner compatibility.
  • Use later Crosstrek pretensioner-style rear belts if required.
  • Reset or replace crash-locked airbag module before final assembly.
Fabricate exhaustWRX exhaust does not naturally terminate like a hatch exhaust.toggle
  • Use WRX J-pipe/downstream path as far as it fits.
  • Fabricate or modify cat-back to fit Crosstrek rear body.
  • Maintain clearance, hangers, heat shielding, and conservative noise.
First start and shakedownRun it stock-ish before adding power.toggle
  • Fill fluids, verify grounds, connect scan tool, confirm immobilizer path.
  • Start and check oil pressure, charging, coolant temperature, fans, throttle, clutch, leaks.
  • Scan codes. Debug lights, locks, hatch, camera, windows, rear wiper, SRS, ABS/VDC.
  • Align, road-test conservatively, then plan suspension/power upgrades.

Electronics workbench

These are the specific electrical problems to solve. This is where the build lives or dies.

Body, cooling, suspension

The drivetrain fit. The cooling/front-end packaging created the major fabrication branch.

Path 1

WRX front-end conversion

Use WRX radiator/front support/front-end components. Remove Crosstrek core support, weld in WRX support structure, fit WRX hood/bumper/headlights, and fabricate hybrid fenders. Reference build used fiberglass strip, rivets, filler, sanding, and paint to merge steel/aluminum fender geometry.

Path 2

Crosstrek nose + custom cooling

Keep Crosstrek exterior front identity and design custom cooling/front-mount intercooler path. Mach V noted this could reduce bodywork and avoid hood conversion, but requires custom charge-air/cooling packaging.

Suspension

Rally/off-road stance

Do not copy lowered reference stance. Select springs/coilovers/control arms after final hub/subframe/brake architecture is known.

Wheels

5x100 vs 5x114

Reference used adapters for VB WRX wheels. Final architecture should decide hubs/brakes/wheels together, not separately.

Exhaust

Hatch-specific rear section

WRX exhaust can be used forward, but the rear section needs custom fitment to the Crosstrek hatch body.

SRS notes

The reference build got stuck here. Treat SRS as a build subsystem, not cleanup.

Known reference issue

Donor airbag module stored a rear passenger crash fault. Rear belts differed between WRX and Crosstrek. WRX used pyrotechnic pretensioners; the Crosstrek rear belt setup did not directly satisfy the WRX module.

Reference solution

Later Crosstrek rear belt units with active pretensioners were sourced and brackets were lightly modified. The airbag module still needed to be removed, sent to a specialist, cleared, and reinstalled.
